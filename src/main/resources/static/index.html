About This Project

What Does This App Do?

A hands-on demonstration of full-stack communication: a browser form talking to a Spring Boot REST endpoint.

🖥️

Frontend

Responsive landing page built with semantic HTML5 and modern CSS3 — flexbox, grid, custom properties, and smooth transitions. No frameworks needed.

⚙️

Backend

Spring Boot 3 REST controller accepts JSON POST requests, validates input, and prints structured output to the IntelliJ console using SLF4J logging.

🔗

Integration

The browser's Fetch API sends an asynchronous JSON request. The server responds with a success / error message that's displayed to the user instantly.

📱

Responsive Design

The layout adapts from wide desktop screens down to 360 px mobile viewports using CSS media queries and a flexible grid system.
